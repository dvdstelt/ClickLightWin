## ClickLight for Windows

Highlight your mouse clicks on screen during live demos, screen sharing, and UX
reviews. ClickLight sits quietly in the system tray and draws a fading pulse
wherever you click, so your audience can follow along.

### Download

Pick one of the assets below:

- **`ClickLight-…-Setup.exe`** (installer, recommended) — installs the app with
  Start-menu and desktop shortcuts, and installs the **.NET 10 Desktop Runtime for
  you if it's missing**.
- **`ClickLight-…-win-x64.exe`** (standalone app, ~250 KB) — for machines that
  already have the .NET 10 Desktop Runtime. Just download and run.

### First run

If Windows SmartScreen shows "Windows protected your PC" on first launch, choose
**More info → Run anyway**.

### What it does

- Fading pulse on click, a contracting ring on hold-and-release, and a drag trail
- **Laser-pointer mode**: a glowing cursor that trails the pointer, plus a fading
  freehand stroke while you drag
- Per-button colors and Size/Duration presets
- A settings window and a dark tray menu with quick toggles
- Global toggle hotkey **Ctrl+Alt+L**, launch-at-login, single-instance guard

### Requirements

Windows 10 21H2+ or Windows 11.

A Windows reimplementation of the macOS app
[ClickLight](https://github.com/aurorascharff/ClickLight).
